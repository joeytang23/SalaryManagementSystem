# SalaryMangerSystem 工资管理系统
基于 C# WinForm + SQL Server 开发的简易企业薪资管理桌面程序

## 项目简介
本系统实现企业员工基础信息录入、薪资核算、工资条生成、薪资数据增删改查、数据导出等核心功能，适合小型公司人事薪资管理使用。

## 技术栈
- 开发语言：C# .NET Framework
- 界面框架：Windows Forms (WinForm)
- 数据库：SQL Server
- 开发工具：Visual Studio 2022
- 版本控制：Git & GitHub

## 功能模块
1. 员工信息管理：新增、删除、修改、查询员工档案
2. 薪资计算：基本工资、绩效、补贴、扣款自动核算
3. 工资单管理：按月生成、查看、打印工资记录
4. 数据操作：薪资记录增删改查、条件筛选
5. 数据库连接：本地SQL数据库持久化存储数据
